AI MARIO GAME

AI Mario Game is a game created by me (Rituparna Das) and the Game Credit goes to Mario. Please read the instruction at the top right hand side to play the game and move your character. You are playing the character of Mario in the game and you need to collect the coins in the game by moving your nose as given in the Instructions. There are some hurdles like the mushroom, spaces in the ground and the pillars. For being safe from the mushrooms, either you can avoid to be in contact with them or jump directly on them to kill them. And you need to jump whenever you observe a space in the platform (ground) or a pillar to avoid a clash with it while your character is running. Click on the Play Game button to start the game. You would get 4 lives in total and after losing all the 4 lives, the game will be over. Please write down your points (the coins collected by you) which will be shown after the game is over. To restart the game, press the Spacebar key on your keyboard. This game is fully based on AI (Artificial Intelligence) as I am using HTML (Hypertext and Markup Language) to design the UI/UX of the web page (game), CSS (Cascading Style Sheets) to style the web page, JS (Java Script) to code for the function and poseNet model under p5.js which is actually ml5.js for this game which is used for detecting the skeleton poses (body parts like the nose, left eye, right eye, left ear, right ear, left shoulder, right shoulder, left elbow, right elbow, left wrist, right wrist, left hip, right hip, left knee, right knee, left ankle and right ankle) and here I am using this model to detect the nose and its X axis and Y axis to move right, left and jump (up). Welcome to the ml5 project! Developing ml5 is not just about developing machine learning software, it is about making machine learning approachable for a broad audience of artists, creative coders, and students. The library provides access to machine learning algorithms and models in the browser, building on top of TensorFlow.js with no other external dependencies.The library is supported by code examples, tutorials, and sample datasets with an emphasis on ethical computing. Bias in data, stereotypical harms, and responsible crowdsourcing are part of the documentation around data collection and usage. We're building friendly machine learning for the web - we're glad you're here! ml5.js is heavily inspired by Processing and p5.js.

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
